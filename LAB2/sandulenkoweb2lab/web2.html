Практика №2
1. Создать удаленный репозиторий
2. Создать локальный проект с помощью npm
3. Подключить локальный проект к удаленному репозиторию
4. Создать html файл содержащий в себе вывод некоторого текста на странице, вывод числа и кнопок, с помощью которых можно увеличивать и уменьшать значение числа на 1.
5. Создать js файл, содержащий в себе логику для работы приложения.
6. В процессе работы над приложением обязательно делать коммиты в удаленный репозиторий.
